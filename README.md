# Personal Finance & Expense Tracker Dashboard

A client-side personal finance dashboard built using HTML, CSS, and vanilla JavaScript.
It allows users to track income and expenses, visualize spending, set budgets, and persist data using localStorage.

## Features

- Add and delete income and expense transactions
- Category-based expense tracking
- Persistent data storage using localStorage
- Income vs Expense bar chart
- Category-wise expense doughnut chart
- Budget rules with visual overspending warnings
- Dark mode with persistent theme preference
- Responsive dashboard layout


## Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Chart.js (for data visualization)
- localStorage (client-side persistence)


## How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
3. No build step or server required
